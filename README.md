# Dilna's Portfolio

A personal portfolio website built with React and Vite, showcasing my work, skills, and contact information.

## Sections

- **Hero** — Introduction and headline
- **About** — Background and skills overview
- **My Work** — Project showcase
- **Contact** — Get in touch form
- **Footer** — Links and closing info

## Tech Stack

- React 19
- Vite 8
- CSS (component-scoped)
- Smooth scroll navigation

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```
