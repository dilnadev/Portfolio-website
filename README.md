A modern and fully responsive personal portfolio website built using React.js
The website is designed to create a strong professional online presence and provide an interactive experience for visitors across all devices.

Features: Responsive modern design, Dark theme UI, Hero section with introduction, About section with skills, Animated skill progress bars, Portfolio/projects showcase,
Contact form section, Smooth and clean user interface, Mobile-friendly layout, Reusable React components.

Technologies Used: React.js, JavaScript, HTML5, CSS3
